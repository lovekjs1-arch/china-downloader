# 모바일 전용 도우인·샤오홍수 다운로더

이 버전은 PC에서 서버를 켜는 방식이 아닙니다.  
Render 같은 클라우드에 한 번 올려놓고, 휴대폰 브라우저에서 인터넷 웹사이트처럼 접속해서 쓰는 구조입니다.

## 왜 그냥 HTML 파일 하나로는 안 되나

도우인·샤오홍수 링크는 단순 이미지 주소처럼 바로 받을 수 있는 구조가 아닙니다.  
브라우저만으로는 사이트의 CORS, 리다이렉트, 서명 URL, 영상 조각 병합, 파일 변환 문제를 처리하기 어렵습니다.

그래서 실제로는 다음 구조가 필요합니다.

휴대폰 웹화면 → 클라우드 서버 → yt-dlp 다운로드 → 휴대폰으로 저장

인터넷에서 보이는 “모바일 웹 다운로더”도 대부분 뒤에 서버가 있습니다.

## 들어간 기능

- 모바일 전용 UI
- 도우인, 샤오홍수, 틱톡 등 yt-dlp가 인식하는 링크 다운로드
- 공유 문구 전체 붙여넣기 가능
- 영상 MP4 저장
- 음성 MP3 추출
- 진행률 표시
- 완료 파일을 휴대폰으로 바로 저장
- 홈 화면 추가 지원
- Android PWA 공유 대상 기본 지원
- 비밀번호 잠금 지원

## Render 배포 방법

### 1. GitHub에 새 저장소 만들기

1. GitHub 접속
2. New repository 클릭
3. 예: `china-mobile-downloader`
4. 이 폴더 안 파일 전체 업로드

### 2. Render에서 Web Service 만들기

1. Render 접속
2. New → Web Service
3. GitHub 저장소 연결
4. Runtime 또는 Language는 Docker 선택
5. 환경변수 추가

필수 환경변수:

```txt
APP_PASSWORD=원하는비밀번호
```

권장 환경변수:

```txt
MAX_LINKS_PER_JOB=5
MAX_AGE_SECONDS=7200
```

### 3. 배포 완료 후 사용

Render가 만들어준 주소 예:

```txt
https://china-mobile-downloader.onrender.com
```

휴대폰에서 접속 → 비밀번호 입력 → 링크 붙여넣기 → 다운로드 시작.

## 휴대폰에서 앱처럼 쓰는 법

Android Chrome 기준:

1. Render 주소 접속
2. 브라우저 우측 상단 메뉴
3. 홈 화면에 추가
4. 앱처럼 실행

## 사용 팁

- 도우인·샤오홍수 앱에서 “공유 → 링크 복사” 후 전체 문구를 그대로 붙여넣어도 됩니다.
- 여러 링크를 줄바꿈으로 붙여넣으면 최대 5개까지 처리합니다.
- 무료 서버는 잠들었다가 깨어날 수 있어서 첫 접속이 느릴 수 있습니다.
- 비공개 영상, 로그인 필요한 영상, 다운로드 제한 영상은 실패할 수 있습니다.
- 사이트 구조가 바뀌면 `yt-dlp` 업데이트가 필요합니다.

## 안전 범위

이 도구는 본인 영상, 편집 권한이 있는 영상, 다운로드가 허용된 공개 영상 저장용입니다.

넣지 않은 기능:

- 워터마크 제거
- 비공개 영상 우회
- 로그인 우회
- 유료 콘텐츠 우회
- 저작권 회피 목적 기능

## 문제 해결

### 다운로드가 실패함

가능한 원인:

1. 링크가 비공개 또는 로그인 필요
2. 도우인·샤오홍수 사이트 구조 변경
3. Render 무료 서버 제한
4. 해당 플랫폼이 서버 IP 접근을 차단

### 파일은 받아졌는데 휴대폰에 저장이 안 됨

브라우저 다운로드 권한을 확인하세요. Android Chrome은 다운로드 폴더에 저장됩니다.

### Render 주소가 남에게 보이면?

반드시 `APP_PASSWORD`를 설정하세요. 비밀번호를 안 넣으면 누구나 접속할 수 있습니다.
